# Ultimate Media Search Bot 🤖💎

Telegram earn bot with Firebase backend & Vercel deployment.

## ⚠️ Security First
1. **NEVER** commit real credentials to public repos
2. Use Vercel Environment Variables:
   - `BOT_TOKEN`: Your Telegram bot token
   - `FIREBASE_DATABASE_URL`: Your Firebase RTDB URL
   - `WELCOME_PHOTO_URL`: https://your-app.vercel.app/static/welcome.jpg
   - `WELCOME_PHOTO_FILE_ID`: (optional) Telegram photo file_id
3. Configure Firebase Rules to restrict write access
